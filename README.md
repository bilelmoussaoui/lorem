# lorem
Lorem Ipsum generator for prototype design

![Lorem Ipsum Generator](https://raw.githubusercontent.com/alt-art/lorem/master/resources/Captura%20de%20tela%20de%202020-11-13%2021-07-29.png)

## Compiling and run

```sh
cargo build
target/debug/lorem
# or
cargo run
```

## Build a .deb

```sh
cargo deb
```
